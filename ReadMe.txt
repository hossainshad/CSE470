# Modern Real Estate Website Color Theme

## Core Colors
- Primary: #1E293B (Deep Navy)
- Secondary: #475569 (Steel Blue)
- Accent: #3B82F6 (Bright Blue)

## Neutral Colors
- Background Light: #F8FAFC (Off White)
- Background Dark: #1E293B (Deep Navy)
- Surface: #FFFFFF (Pure White)

## Text Colors
- Text Primary: #1E293B (Deep Navy)
- Text Secondary: #64748B (Muted Blue)
- Text Light: #F8FAFC (Off White)

## UI Elements
- Border: #E2E8F0 (Light Gray)
- Input Focus: #3B82F6 (Bright Blue)
- Error: #EF4444 (Red)
- Success: #10B981 (Green)

## Gradients
- Primary Gradient: linear-gradient(145deg, #1E293B, #475569)
- Accent Gradient: linear-gradient(to right, #3B82F6, #2563EB)

## Usage Guidelines

### Header & Navigation
- Background: Primary (#1E293B)
- Text: Light (#F8FAFC)
- Active/Hover: Accent (#3B82F6)

### Content Areas
- Background: Light (#F8FAFC)
- Cards: Surface (#FFFFFF)
- Shadows: rgba(0, 0, 0, 0.1)

### Buttons
- Primary: Accent (#3B82F6)
- Secondary: Surface (#FFFFFF) with border
- Text on Dark: Light (#F8FAFC)

### Forms
- Input Background: Surface (#FFFFFF)
- Input Border: Border (#E2E8F0)
- Focus: Accent (#3B82F6)

## Accessibility
- Maintain WCAG 2.1 AA standard
- Minimum contrast ratio: 4.5:1
- Focus states clearly visible
